### დავალება

შეიმუშავეთ Java პროგრამა, რომელიც გამოიყენებს HashMap კლასს სტუდენტების მონაცემთა ბაზის შესაქმნელად.
გამოიყენეთ სტუდენტის ID, როგორც გასაღებები და Student class როგორც მნიშვნელობების ობიექტი, რომელსაც ექნება სახელი, გავარი, ჯგუფი, ნიშნების სია.
განახორციელეთ ფუნქციები HashMap-დან სტუდენტის დასამატებლად, ამოსაღებად და მოძებნისთვის.
გაფილტრეთ ისეთი top 3 სტუნენტები, რომლებსაც ნიშნების სიაში ყველაზე მაღალი ქულა გამოზდით.
